# Invoke-WslDebugShell

## SYNOPSIS

Invoke WSL system debg shell.

## SYNTAX

### __AllParameterSets

```text
Invoke-WslDebugShell [<CommonParameters>]
```

## ALIASES

There are no aliases for this cmdlet.

## DESCRIPTION

Invokes the WSL system debug shell.

## EXAMPLES

### EXAMPLE 1

```powershell
PS > Invoke-WslDebugShell
```
